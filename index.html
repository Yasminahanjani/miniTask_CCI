HTML

Ayo mengenal apa itu HTML!

Apa sih HTML itu?

HTML merupakan singkatan dari Hypertext Markup Language yaitu bahasa standar web yang dikelola oleh W3C (World Wide Web Consortium) yang berisi tag-tag untuk menyusun tiap elemen dari website. HTML berperan penting dalam membentuk struktur halaman website dan menempatkan elemen sesuai dengan desain maupun keinginan dari pembuat web itu sendiri.


Apa saja unsur-unsur dari HTML?

Tag

Tag adalah simbol khusus (Markup) berupa dua karakter "<" dan ">" yang berada di awal dan akhir dari suatu teks sebagai nama tag. Tag biasanya berguna untuk membatasi pengaruh ataupun bagian dari nama tag tersebut. Tag yang paling umum ditemui dalam HTML adalah tag body yang ditulis < body >. Tag tersebut digunakan untuk menandakan awalan dari suatu bagian. Sedangkan untuk menutupnya menggunakan tag yang diimbuhi "/" pada awal nama tag, contohnya < /body >. Beberapa karakterisitik dari tag antara lain:

1. Tag bersifat incase sensitive, sehingga boleh ditulis menggunakan huruf besar(kapital), huruf kecil, maupun campuran dari huruf besar dan kecil sekaligus. Namun pada umunya, tag ditulis menggunakan huruf kecil semua.
2. Tag mungkin diisi oleh tag lain. Maksudnya adalah, di dalam suatu tag dapat terdiri dari banyak tag lain. Tag-tag tersebut dimasukkan ke dalam suatu tag induk, untuk menandakan bahwa ia berada di dalam bagian dengan nama tag tersebut.

Atribut

Atribut atau disebut juga sebagai properti digunakan untuk mengatur bagaimana elemen dari suatu tag akan ditampilkan. Atribut ditulis di dalam penulisan tag, setelah menuliskan nama tag dengan dipisahkan oleh spasi. Contohnya adalah tag < p style="text-align: center" >. Arti dari atribut tersebut adalah, tag paragraf akan ditampilkan dengan susunan kalimat berada di tengah. Contoh lain dari penulisan atribut adalah untuk penulisan class maupun id, < section class="container" >. Beberapa karakteristik dari atribut adalah sebagai berikut:

1. Atribut hanya ditulis pada tag pembuka saja.
2. Setiap tag memiliki atribut yang berbeda-beda.
3. Beberapa atribut ada yang disebut dengan global attribute karena dapat diterapkan ke banyak tag lain, contohnya seperti atribut class, id, dan style.
4. Setiap tag dapat ditulis dengan banyak atribut, satu atribut, bahkan tidak sama sekali.
5. Jika tag berisi banyak atribut, tiap atribut dipisahkan oleh spasi.

Elements

Element adalah bagian dari skrip HTML yang terdiri dari tag pembuka. isi element, dan tag penutup. Contoh dari elemen adalah < p > Hello, World! < /p >. Kata Hello, world! yang tampil pada page website merupakan elemen dari HTML tersebut.


Lebih tahu tentang struktur HTML.

HTML disimpan dengan ekstensi ".html". Jika di dalamnya terdapat skrip PHP, maka ekstensinya adalah ".php". Dalam penulisan HTML, ada beberapa tag yang wajin dituliskan untuk membangun website html yang baik dan terstruktur. Tag tersebut adalah sebagai berikut:

  • < !DOCTYPE HTML >, dimana ini merupakan tag awal untuk membuat dokumen HTML. Fungsinya adalah untuk memberi tahu browser bahwa dokumen yang sedang dibuka ini adalah dokumen HTML.

  • < html >< /html >, adalah tag yang menunjukan tanda untuk membuka dan menutup dokumen html.

  • < head >< /head >, tag ini digunakan untuk menyimpan berbagai informasi tentang dokumen HTML. Semua elemen yang ada di tag ini tidak akan ditampilkan ke browser.

  • < title >< /title >, adalah tag yang dugunakan untuk membuat judul website yang dimunculkan pada title browser.

  • < body >< /body >, merupakan tag yang menunjukkan bagian utama dari website. Semua yang akan ditampilkan pada website dan halaman browser ditulis dan diletakkan di tag ini.

CSS

Ayo mengenal apa itu CSS!

Apa sih CSS itu?

CSS adalah singkatan dari Cascading Style Sheet yaitu dokumen web yang berfungsi untuk mengatur elemen HTML dengan berbagai properti yang tersedia sehingga dapat ditampilkan dengan berbagai gaya yang diinginkan. Cara kerja CSS dalam memodifikasi HTML dengan memilih elemen HTML mana yang akan diatur, kemudian memberikan properti yang sesuai dengan desain maupun tampilan yang diinginkan. Skrip CSS terdiri dari 3 bagian, yaitu selector untuk memilih elemen HTML yang ingin dimodifikasi, properti adalah aturan yang akan diberikan, dan value adalah nilai dari aturan yang diberikan.


Bagaimana cara menuliskan CSS?

Pada dasarnya, penulisan CSS dibagi menjadi 3 cara. Cara-cara tersebut antara lain inline, internal, dan external. Ketiganya dapat digunakan sesuai dengan kebutuhan.

Inline

Inline CSS berarti menuliskan CSS dengan menggunakan atribut style yang dituliskan secara langsung pada dokumen html setelah nama tag yang dipilih. Penulisan dengan cara ini cocok untuk menuliskan CSS yang hanya memiliki satu atau dua properti saja dan bersifat permanen. Kekurangan dari penulisan CSS ini adalah susah ketika terjadi perbaikan desain, karena harus memperbaiki seluruh inline css yang ada.

Contoh dari penulisannya adalah :
< p style="color: yellow" >
< container style="margin-top:20px">

Internal

Internal CSS yaitu penulisan CSS yang menggunakan tag < style >< /style > dan diletakkan di dalam elemen < head >....< /head >. Cara penulisan ini cocok apabila CSS ini hanya digunakan untuk halaman HTML itu saja. Artinya tidak ada file html lain yang menggunakan CSS tersebut. Penulisan internal CSS jyga digunakan apabila CSS hanya memiliki sedikit selector. Kekurangan dari penulisan ini adalah CSS tidak dapat diterapkan di halaman HTML lainnya.

Contoh dari penulisannya adalah :
< style >
p {
color:"red";
font-size:"15pt";
}
< /style >

External

External CSS adalah penulisan CSS yang disimpan di halaman berbeda dari file HTML, dan disimpan dengan ekstensi ".css". Untuk menghubungkan file HTML dengan CSS, file CSS dipanggil di dalam elemen < head > dengan cara < link rel="stylesheet" href="namafile.css" >. Nama file CSS pada atribut href disesuaikan dengan letak file CSS disimpan. Cara penulisan ini paling disarankan dalam pembuatan website menggunakan HTML. Karena file CSS dapat digunakan oleh lebih dari satu halaman HTML hanya dengan cara dipanggil saja, hal ini meningkatkan efisiensi waktu dalam pembuatan website.


Apa saja selector pada CSS?

Selector adalah sebuah kata yang digunakan untuk memilih elemen HTML tertentu yang akan dimodifikasi. Selector juga digunakan sebagai penanda bahwa perubahan atau pengaturan gaya dalam dokumen HTML hanya berlaku untuk elemen yang dipilih saja. Secara garis besar, ada 3 elemen utama dalam penulisan selector CSS, yaitu:

1. Selector tag, yaitu selector yang langsung menyebutkan nama tag HTML yang dipilih untuk diatur menggunakan CSS. Penulisan dalam CSS juga cukup menyebutkan nama tagnya saja, seperti h2, p, dan body.
2. Selector Class, yaitu selector yang menyebutkan nama class dari suatu tag yang ingin dimodifikasi menggunakan CSS. Pemanggilan atau penyebutan nama class dalam css diimbuhi dengan tanda (.) sebelum nama classnya. Contohnya adalah, .container, .navbar, dan .main-content.
3. Selector Id, sama seperti selector class, di selector id cukup dengan memanggil ataupun menyebut nama id dari suatu tag saja. Penulisan nama id dalam css diimbuhi dengan simbol (#). Contoh penulisannya adalah #content-info, #boldTitle, dan #setRed.

Javascript

Ayo mengenal apa itu Javascript!

Apa sih Javascript itu?

Javascript merupakan bahasa pemrograman web yang pemrosesannya dilakukan di sisi client. Jadi, javascript hanya dapat dijalankan dengan menggunakan browser. Javascript dapat dijalankan ketika ada event tertentu yang terjadi pada halaman web. Baik event yang dilakukan oleh user, maupun event yang terjadi karena adanya perubahan paa halaman website.


Bagaimana cara Meletakkan Javascript pada file HTML?

Ada berbagai cara untuk meletakkan kode javascript pada dokumen HTML. Diantaranya adalah:

Menggunakan Event-handler (Inline)

Cara ini biasanya digunakan ketika kode javascript hanya sedikit, ataupun hanya sekedar untuk memanggil function saja. Cara meletakkannya adalah dengan menambahkan kode setelah atribut event pada tag HTML.

Contohnya adalah :
< button onclick=alert('Anda menambahkan data')" > Tambah data < /button >

Menggunakan Tag < script > (Internal)

Penulisan Javascript ini dilakukan apabila kode javascript yang ditambahkan hanya terdiri dari beberapa baris dan tidak terlalu panjang, sehingga cukup untuk dimasukkan ke dalam dokumen HTML. Umumnya penulisan javascript dengan cara ini dilakukan dengan meletakkannya di bawah dokumen, sebelum tag body tutup < /body >. Namun, javascript dapat diletakkan di baris mana saja.

Contoh dari penulisannya adalah :
< script >
alert("Selamat Datang!");
< /script >

Menggunakan Tag < script src=".." > (External)

Penulisan javascript ini dilakukan pada kode javascript yang sangat panjang. Javascript tersebut disimpan dalam file tersendiri dengan ekstensi ".js". File tersebut kemudian dipanggil dengan cara < script src="namafile.js" >< /script>.


Mengenal dasar-dasar dalam Javascript

  • Variabel
    Variabel digunakan untuk menyimpan data berupa teks, angka, dan lainnya dimana nilai dari variabel dapat diubah-ubah. Aturan penulisan dalam variabel adalah sebagai berikut:
    1. Jangan memberikan nama variabel berupa kata-kata yang dapat menjadi kode javascript. Contohnya seperti break, boolean, else, and, var, dan masih banyak lagi.
    2. Tidak boleh diawali dengan angka atau simbol, penulisan variabel setidaknya diawali dengan huruf atau underscore.
    3. Variabel pada javascript bersifat case-sensitive, sehingga penulisan huruf besar dan kecil harus diperhatikan.
  • Tipe Data
    Pada javascript, tipe data bersifat seolah-olah menjadi objek. Sehingga setiap variabel pada javascript akan langsung memiliki method. javascript mengenal 3 tipe data dasar, yaitu numbers, strings, dan boolean. Selain itu, javascript juga mengenal tipe data null dan undefined.
  • Operator
    Operator merupakan sebuah simbol atau kata yang digunakan untuk melakukan operasi terhadap satu atau lebih daya atau variabel untuk menghasilkan data baru. Variabel atau data baru yang diperoleh tersebut disebut dengan operand. Beberapa jenis operator adalah sebagai berikut:

    Operator Aritmatika

    Operator ini digunakan dalam perhitungan data numerik. Beberapa bentuk operator ini adalah sebagai berikut:
    + : Penjumlahan
    - : Pengurangan
    * : Perkalian
    / : Pembagian
    % : Modulus (Sisa Hasil Bagi)
    ++ : Penjumlahan dengan 1
    -- : Pengurangan 1

    Operator perbandingan

    Operator ini digunakan untuk membandingkan dua variabel atau data. Hasil perbandingan ini akan bernilai true atau false. Berikut adalah beberapa bentuk operator perbandingan:
    == : Sama dengan
    != : Tidak sama dengan
    > : Lebih dari
    < : Kurang dari
    >= : Lebih dari sama dengan
    <= : kurang dari sama dengan

    Operator Logika

    Operator ini digunakan untuk mengoperasikan data boolean. Hasil dari operator ini juga akan bernilai true dan false. Berikut adalah beberapa bentuk operator logika:
    && : Menghasilkan nilai true apabila kedua variabel bernilai true
    || : Menghasilkan nilai true apabila salah satu variabel bernilai true
    ! : Mengembalikan nilai kebalikan atau bukan dari kondisi variabel sekarang (is not)

    Operator Assignment

    Operator ini digunakan untuk memberikan nilai kepada suatu variabel menggunakan tanda sama dengan. Berikut adalah beberapa bentuk operator Assignment:
    = : Memberi nilai suatu variabel
    += : Menambah nilai suatu variabel
    -= : Mengurangi nilai suatu variabel
    *= : Mengalikan nilai suatu variabel /= : Membagi nilai suatu variabel
    %= : Mendapatkan hasil (Modulus) bagi dari suatu variabel

    Operator Kondisi

    Operator ini digunakan untuk memberikan nilai pada suatu variabel berdasarkan kondisi yang ditentukan. Ditandai dengan penulisan ? ... : ..... Apabila suatu kondisi yang dituliskan sebelum tanda tanya bernilai true, maka akan mengeluarkan nilai yang berada di sebelah kiri tanda titik dua(:). Sedangkan ketika menghasilkan nilai false, maka akan menghasilkan nilai yang berada di sebelah kanan tanda gtitik dua(:).


  • Function
    Function merupakan blok program yang dapat digunakan secara berulang-ulang. Penggunaan function pada kode program hanya dengan pemanggilan nama functionnya saja. Function memungkinkan program untuk mengembalikan nilai return maupun tanpa dengan membalikan nilai. Penulisan function pada javascript adalah sebagai berikut:
    function namafungsi(parameter1, parameter2) {
    pernyataan;
    return variabel;
    }

  • Event
    Event merupakan suatu tindakan yang dilakukan terhadap suatu elemen HTML. Ada berbagai macam event, sebagai berikut:
    onblur : ketika suatu elemen tidak fokus
    onclick : ketika suatu elemen diklik
    ondblclick : Ketika suatu elemen didouble klik
    onkeydown : Ketika tomboh huruf di keyboard ditekan
    onkeyup : ketika tombol keyboard dilepas
    onkeypress : Ketika tombol keyboard ditekan
    onload : Ketika dokumen diload
    onselect : Ketika elemen dipilih
    onsubmit : Ketika data form dikirim
    dan masih banyak lagi.
